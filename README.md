# OSDU Schema Tree Visualizer

**A clean, layered tree visualizer for the OSDU data model universe.**

This repo was created to provide a simple Streamlit app that renders OSDU schemas as your requested tree: Reference Data (roots) → Master Data (stem) → WPC (branches) → Datasets (leaves).

## Quick Start

1. Clone schemas: `git clone https://community.opengroup.org/osdu/data/data-definitions.git`
2. `pip install streamlit networkx plotly pandas`
3. Run `streamlit run osdu_tree_viz.py` (after cloning this repo)
